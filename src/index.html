Proyectos

Around the U.S.

2025 - Bootcamp TripleTen

Descripción

Around the U.S. es una aplicación web interactiva y responsiva desarrollada con JavaScript orientado a objetos, que consume una API REST mediante fetch y manejo de asincronía. Implementa renderizado dinámico con templates, arquitectura modular con clases reutilizables, y formularios con validación personalizada para la gestión de contenido en tiempo real.

Tecnologías utilizadas

  • HTML5
  • CSS3 (BEM)
  • JavaScript ES6+ (OOP, Fetch, Asincronía)
  • APIs REST
  • Modularización
  • Diseño Responsive

Funciones Clave

  • Edición dinámica de perfil con actualización persistente en API
  • Creación y eliminación de tarjetas con renderizado dinámico
  • Integración con API REST (GET, POST, PATCH, DELETE)
  • Implementación de popups modales reutilizables
  • Validación de formularios con clases personalizadas
  • Arquitectura basada en POO y modularización del código
  • Manejo de asincronía con Promises

Art Gallery

2024 - Bootcamp TripleTen

Descripción

Art Gallery es una aplicación web responsiva enfocada en la construcción de layouts modernos utilizando CSS Grid y Flexbox, organizada bajo la metodología BEM para mantener un código modular y escalable. El proyecto prioriza buenas prácticas de estructura y diseño frontend adaptable a múltiples dispositivos.

Tecnologías utilizadas

  • HTML5
  • CSS3 (CSS Grid, Flexbox, BEM)
  • Diseño Responsive

Funciones Clave

  • Estructura visual avanzada con CSS Grid para organización de contenidos
  • Alineación y distribución responsiva con Flexbox
  • Estilizado modular y consistente usando metodología BEM
  • Diseño adaptativo para múltiples dispositivos
  • Código organizado para fácil mantenibilidad
