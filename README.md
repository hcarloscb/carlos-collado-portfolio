# Portafolio – Carlos Collado

Portafolio personal desarrollado para presentar mis proyectos como Frontend Developer.

El sitio está construido con HTML y CSS, con enfoque en estructura semántica, organización modular y claridad visual. Esta primera versión está orientada a consolidar la base visual y estructural del proyecto antes de integrar mejoras progresivas como responsividad y funcionalidades dinámicas.

---

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- Metodología BEM
- Git & GitHub

---

## 🧩 Estructura del proyecto

El portafolio está dividido en:

- **Sobre mí** – Perfil profesional y enfoque técnico
- **Proyectos** – Aplicaciones desarrolladas durante mi formación
- **Contacto** – Información para conexión profesional

Se aplicaron principios de separación de responsabilidades y organización clara de clases CSS.

---

## ✨ Características técnicas

- Estructura HTML semántica
- Organización CSS bajo metodología BEM
- Jerarquía visual clara
- Micro-interacciones mediante transiciones CSS
- Código estructurado para facilitar futuras mejoras (responsividad y JavaScript)
